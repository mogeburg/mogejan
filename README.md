## 開発向け情報

### バージョンについて

- パッチバージョンアップ時は末尾に1を追加していく（なぜ？）  
  満足したらマイナーバージョンアップとして小数点第一を上げること
  - `v0.01` -> `v0.011` -> `v0.0111` -> ... -`> v0.02`

- メジャー・マイナーバージョンを上げる際は、先に[タグを切ること](https://github.com/mogeburg/mogejan/releases/new)
  - `Tag:Select tag` -> `Create new tag` -> `vX.XX`
  - `Release Title`はタグ名そのままで👌
  - `Release notes`は`Generate release notes`そのままで👌
  - 問題なさそうなら`Publish release`する

### 特殊能力表

- あいもげ: AI is Watching You（リーチされた時に自動発動）
  - 手牌の中にある危険牌がわかるようになる
- ぶるもげ: もうあった！ｗｗ （自分の捨牌に上がり牌がある、かつリーチした時に自動発動）
  - 自分の捨牌に当たり牌がある場合、次の自分のツモと入れ替えて引く
- みいもげ: み！ （局開始時に自動発動）
  - 他プレイヤーは手牌時点で5役以上ないとリーチができなくなる
- ピカさん: そうだね （リーチした時に自動発動）
  - 能力発動後に自分が上がった場合、特殊役そうだね(3役)が追加される
- しらん女: 知らん （ツモ・ロンされた時に自動発動）
  - 他プレイヤーからツモ・ロンされた時に自分が引かれる点数を0にする
- あの娘: 一緒にうまれたのに （ツモ・ロンした時に自動発動）
  - あの娘の牌を知らん女として、知らん女の牌をあの娘の牌としても扱い役計算を行う
  - 例: 知らん女,あの娘,妹 の場合 -> 知らん女,あの娘,妹 と あの娘,あの娘,妹 と 知らん女,知らん女,妹 の3種類で計算を行い、重複していない役をすべて付与する
- 妹: ちっちゃい悪いこと （リーチしていない、かつリーチされた時に自動発動）
  - リーチした人の手牌と自分の手牌をすべて入れ替える
- あねもげ: dice1d8 （局開始時に自動発動）
  - 8面ダイスが振られ、出た目の数だけ手牌にあねもげの牌を引く
- おちゃんち: 召喚札 （局開始時に自動発動）
  - 一部の基本牌をツモしやすくなる（ぶるもげ、みいもげ、知らん女、あの娘、妹、おちゃんち）
